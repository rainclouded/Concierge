Incident Reports

View Incident Reports

query Parameters
severities
string
Example: severities=LOW,HIGH

A comma-separated list of severity values to filter the reports. Valid values: LOW, MEDIUM, HIGH, CRITICAL.

statuses
string
Example: statuses=OPEN,CLOSED

A comma-separated list of status values to filter the reports. Valid values: OPEN, CLOSED, RESOLVED, IN_PROGRESS.

beforeDate
string <date>
Example: beforeDate=2023-10-01

A date string to filter reports created before this date (format: YYYY-MM-DD).

afterDate
string <date>
Example: afterDate=2023-09-01

A date string to filter reports created after this date (format: YYYY-MM-DD).
